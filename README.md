# Jethro's Dotfiles

## Prerequisits

zsh, rg, fzf

* install rbenv https://github.com/rbenv/rbenv-installer#rbenv-installer
    * then install ruby


* install nodejs (I prefer nvm https://github.com/nvm-sh/nvm#install--update-script to install node)

## Installing

Run the install script:

`zsh install.sh`

Coc setup
* Open vim and do 
  * :CocInstall coc-java
  * :CocInstall coc-pyright
  * :CocInstall coc-solargraph
